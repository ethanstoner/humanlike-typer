# HumanType v1.1.2

This release focuses on making the native Windows app feel production-ready.

## Added

- GitHub Releases update checking inside the Windows app.
- `Check Updates` and `Release Notes` controls in Settings.
- `Check for Updates` in the tray menu.
- Release notes dialog that shows GitHub release notes in-app.
- One-time release notes display after the installed version changes.

## Improved

- Sliders now support direct typed values as well as dragging.
- Slider handles no longer clip at maximum values.
- Slider tracks no longer show extra edge artifacts.
- Closing the settings window with the X button now hides cleanly to tray without flashing.
- Windows build metadata now uses the packaged app version for updater comparisons.

## Release Assets

For Windows users, download and run:

- `HumanType-Installer.exe`

Exit any running HumanType instance from the tray before installing the update.
